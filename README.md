Este repositório contém o código-fonte do meu portfólio profissional de desenvolvimento Front-end. Projetado com **estética editorial minimalista**, alto rigor tipográfico e movimento fluido, o projeto materializa a união de mais de **25 anos de carreira no mercado criativo** (Motion Design, Direção de Arte e UX/UI) com o desenvolvimento moderno de software.

O resultado é um espaço digital de alta performance que atua como a ponte perfeita entre o design conceitual refinado e a engenharia de software escalável.

---

## 🎯 // 01. O DIFERENCIAL DO PROJETO

Diferente de portfólios genéricos, este projeto foi concebido sob a filosofia de que **"a preciosidade reside nos detalhes"**. Ele reflete a fusão de duas mentalidades:

1. **O Olhar Clínico do Designer:** Valorização obsessiva do espaço em branco (respiros), entrelinhas impecáveis, contraste tipográfico severo e animações orgânicas inspiradas na paciência e no cuidado milimétrico aplicados no cultivo clássico do **Bonsai**.
2. **O Rigor Técnico do Desenvolvedor:** Padrões de código limpo (*Clean Code*), desenvolvimento de componentes modulares reaproveitáveis (*Design Systems*), acessibilidade inclusiva **WCAG 2.1 AA** e otimização agressiva de desempenho para **Core Web Vitals**.

---

## ⚙️ // 02. ECOSSISTEMA E STACK TÉCNICA

Construído com tecnologias topo de linha do desenvolvimento web contemporâneo:

- **Framework:** [Next.js 16 (App Router)](https://nextjs.org/) (Compilação e roteamento estático ultra velozes via Turbopack)
- **Biblioteca Base:** [React 19](https://react.dev/)
- **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/) (Estilização de alta performance com variáveis nativas no `@theme`)
- **Movimento e Micro-interações:** [Framer Motion](https://www.framer.com/motion/) (Física de aceleração natural `[0.16, 1, 0.3, 1]`)
- **Iconografia:** [Lucide React](https://lucide.dev/) + SVGs inline oficiais para compatibilidade de marcas.
- **Tipografia:** Plus Jakarta Sans (Headers elegantes) e JetBrains Mono (Dados técnicos de Grid).

## ✨ // 03. RECURSOS EXCLUSIVOS E ENGENHARIA VISUAL

### 🌌 A. Hero Section de Faceta Dupla (Dual-Facet Switcher)
O visitante pode alternar dinamicamente a interface do site entre duas facetas:
- **`// CREATIVE.UX`:** Destaca a bagagem artística com um título fluído e a renderização de um **Bonsai vetorial orgânico em SVG**, cujas folhas pulsam via animações CSS delicadas.
- **`// DEV.FRONTEND`:** Chaveia para o foco técnico em código limpo, renderizando um painel editor de código interativo em TSX de alta fidelidade visual.

### 🎯 B. Cursor Magnético com Física Fluida (Lerp 15%)
Desenvolvido em Javascript puro com movimentação controlada via `requestAnimationFrame` e transforms `translate3d` em 60fps, o cursor apresenta um ponto central com um anel externo que desliza e se expande magneticamente sobre botões e links. O cursor auto-detecta dispositivos touch screen e se desativa graciosamente em celulares.

### 📐 C. Grade Editorial Assimétrica (Asymmetric Project Grid)
Em substituição aos grids simétricos tradicionais, a seção de projetos adota um layout editorial dinâmico de revista (editorial design), alternando proporções e larguras de cards dependendo do destaque do projeto.

### ➕ D. Modal com Accordions Otimizados (Anti-Wall-of-Text)
Inspirado por técnicas modernas de simplificação visual, o modal de detalhes divide-se em duas colunas (Mídia à esquerda / Metadados à direita). As explicações de *Desafio*, *Solução* e *Ficha Técnica* são recolhidas em accordions interativos com transição de altura dinâmica e rotacionamento de ícones.

### 🤖 E. SEO Técnico & AI Crawler Ready
Configurado com dados estruturados da Schema.org (`Person`) e metadados de robots de profundidade máxima (`max-image-preview: "large"`), garantindo que tanto buscadores tradicionais (Google) quanto assistentes de busca de IA (Google Gemini, SearchGPT, Perplexity e OpenAI) indexem o portfólio com resumos ricos e em alta definição.

## 🚀 // 04. INSTALAÇÃO E USO LOCAL

Para clonar e testar o portfólio localmente:

1. **Clonar o Repositório:**
   ```bash
   git clone https://github.com/dimildesigner/meu-portfolio.git
   cd meu-portfolio
   ```

2. **Instalar Dependências:**
   ```bash
   npm install
   ```

3. **Executar Servidor de Desenvolvimento:**
   ```bash
   npm run dev
   ```
   Acesse [http://localhost:3000](http://localhost:3000) no seu navegador.

4. **Compilar para Produção (Build de Produção):**
   ```bash
   npm run build
   ```
   O Next.js gerará uma distribuição estática ultra otimizada e minificada na pasta `.next`.
